## Web Portfolio – Dagmar Vodáková

Osobní webové portfolio sloužící k prezentaci projektů, technických dovedností a profesního profilu.
Web je publikován pomocí **GitHub Pages** a funguje jako centrální rozcestník na mé GitHub repozitáře a samostatné projekty.

🌐 **Online verze:** https://dagydee.github.io

---

### Obsah

- **Index** – stručné představení zaměření a profilu
- **Projekty** – přehled vybraných Python a webových projektů s technickým popisem a odkazy na repozitáře
- **Dovednosti** – používané technologie, nástroje a oblasti zaměření
- **Vzdělání & kurzy** – relevantní kurzy a samostudium v oblasti IT
- **CV** – online verze životopisu + ke stažení v PDF
- **Kontakt** – kontaktní údaje a odkazy na GitHub / LinkedIn

### Použité technologie

- **HTML5** – sémantická struktura
- **CSS3** – responzivní layout (Flexbox, Grid)
- **Git & GitHub Pages** – verzování a statická publikace

Web je koncipován jako **statický**, bez použití backendu nebo frameworků, s důrazem na:
- čitelnost a udržitelnost kódu
- jednoduchou strukturu
- dobrý výkon a rychlé načítání

### Cíl projektu

- praktické procvičení tvorby vícestránkového statického webu
- prezentace vlastních projektů a technických dovedností
- vytvoření přehledné online vizitky pro technické i HR účely
